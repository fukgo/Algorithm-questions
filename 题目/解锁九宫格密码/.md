# 强强连手-你好cmen

## 题目背景

**你怎么敢打扰那位大人?!!**  
-某位深高级学长

## 题目描述

深圳职业技术大学人才辈出，其中包括无所不能的cmen。  

现在你要去找他的宿舍和他强强联手毕业项目 。可是他的宿舍门有一个九宫格手势密码。现在他告诉你密码 

选择请以字符矩阵（0<=矩阵字符<=9，1为开始滑动，结束滑动数据小于等于9）模拟滑动解锁的过程（按照序号滑动顺序）
如果全是0那么输出no

## 输入格式

输入一个3x3的字符矩阵并输出你设的密码方向
比如  
001  
204  
000  
解锁坐标顺序是(3,1) -> (1,2) -> (2,2) -> (3,2) 

只考虑最短的连点  

因为中间的点是最短的所以会碰到  

题目答案请使用坐标系坐标表示法(x,y)  

而不是数组表示法

## 输出格式

(3,1) -> (1,2) -> (2,2) -> (3,2)

## 样例 #1

### 样例输入 #1

```
001
204
000
```

### 样例输出 #1

```
(3,1) -> (1,2) -> (2,2) -> (3,2)
```

## 提示

别忘了空格！  
不保证 一定有大于0的字符  
一个点可能被重复划到只需要打印出第一次划到的坐标
